# Fusion Auth NodeJS
#### This is just a testing application using Fusion Auth

### Tech Stacks
- Node v12+
- Express
- Nunjucks
- Fusion Auth

### Installation
1. Run `npm install`
2. Make sure nodemon installed to watch all files are changing in your code. If you don't have please install it `npm install -g nodemon`
3. Run `npm start`
